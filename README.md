## 📱 How to Install on Phone (No App Store needed)

**Android (Chrome):**
1. Open the link above in Chrome
2. Tap ⋮ Menu → **"Add to Home Screen"**
3. Tap **"Add"**
4. App icon appears on your home screen ✅

**iPhone (Safari):**
1. Open the link above in Safari
2. Tap the **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**
4. Tap **"Add"**
5. App icon appears on your home screen ✅

## ⚙️ Features

| Feature | Description |
|---|---|
| 📷 Scan QR Code | Scan equipment QR code with camera |
| 🖼️ From Gallery | Pick QR code image from phone gallery |
| 📂 Load History | Open Google Drive folder with all equipment files |
| 📜 Repair History | View full service history of any equipment |
| ➕ Add Record | Add new repair/service entry |
| 🔐 Admin Panel | Edit or delete records (Password: authorized users only) |

---

## 🔧 Equipment Details Tracked

- Equipment Name, ID, KW, HP, RPM
- Frame Size, DE Bearing, NDE Bearing
- Serial Number
- Task Performed, Date, Status
- Completed By, Edited By
